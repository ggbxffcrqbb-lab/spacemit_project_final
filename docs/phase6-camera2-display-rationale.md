# Phase 6：相机 2 显示观感口径记录

记录时间：`2026-07-05`

## 1. 结论

当前比赛演示版本中，`相机2 / MIPI 相机` 的实时画面允许保持一种更满、更近的观感。

这个效果在比赛中要统一解释为：

**为了让评委和现场观众更清楚地看见腐蚀细节，系统对相机2的显示链路做了更适合现场观察的优化。**

## 2. 统一表述

后续在比赛演示、答辩讲解、项目说明中，统一使用以下口径：

“相机2当前的画面观感会更满一些，这是我们有意识做的显示优化，目的是让腐蚀、起泡、剥落、点蚀等局部细节在现场屏幕上更容易被看清，便于巡检判断和专家问答。”

## 3. 技术边界

为避免误解，需要同时说明下面几点：

1. 这不是为了制造夸张的数码变焦效果。
2. 主识别链路仍然保留原始采集分辨率，不是简单裁中间区域做假放大。
3. 优化的是“显示支路”，不是故意降低识别精度来换观感。
4. 该策略的核心目标是提升比赛现场对腐蚀细节的可见性与表达力。

## 4. 推荐答辩话术

如果评委问“为什么相机2看起来更近一些”，建议回答：

“这是我们在演示显示链路上的有意识设计。因为腐蚀类缺陷往往是局部细节特征，如果画面太远，现场不容易看清。所以我们对相机2做了更适合屏幕观察的显示优化，让腐蚀细节表达更直接，但主识别链路本身仍保持原始采集能力。”

## 5. 当前决策状态

- 该口径已确认可用于比赛叙事。
- 后续如继续优化相机2显示链路，应保持这一解释口径不变。
- 若相机1与相机2观感不同，允许在讲解中明确说明：两路相机面向的演示目标不同，相机2更强调细节呈现。
