# 防腐知识库种子说明

这里存放当前板端离线 RAG 的**首批本地知识卡**。

当前定位：

1. 用于把作品从“通用语音助手”升级为“能引用本地资料的防腐问答终端”。
2. 先支撑 Phase 3 + 最小可用 RAG，不冒充完整国标或企业规程库。
3. 后续可继续补充标准、SOP、案例、巡检记录和企业内部手册。

当前这些文档以**工程演示知识卡**为主，适合做：

- 涂层起泡、开裂、脱落、锈蚀等缺陷初判
- 储罐和管道外壁腐蚀巡检问答
- 高风险场景下的保守处置建议

不适合直接替代：

- 设计计算
- 承压设备定级
- 动火、受限空间、停输停产等审批决策
- 企业正式工艺卡和强制性标准文本
